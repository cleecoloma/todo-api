# TO DO API

> Integrating with a live API with the To Do App (Frontend)

## Installation

> Start with: `npm install`

## Usage

Signup request:
```text
method: POST
route: /signup
data: {
  username: {ENTER_NAME}
  password: {ENTER_PASSWORD}
  role: {PICK_ROLE}
}
```

Signin request:
```text
method: POST
route: /signin
headers: Authentication Basic {
  username: {ENTER_NAME}
  password: {ENTER_PASSWORD}
}
```
